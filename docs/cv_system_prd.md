# 简历智能润色系统 - 任务确认单

## 1. 核心目标
构建一个智能化的简历修改与润色系统，解决使用 HTML 编写简历时不够直观、修改繁琐的问题，并结合大语言模型提供专家级的简历优化建议与自动修改能力。

## 2. 角色定位 (AI)
- 顶级简历修改专家
- 资深职业规划专家

## 3. 核心功能及交互流程

### 3.1 页面布局 (三栏式设计)
*   **左侧 (AI 交互区):**
    *   **输入功能:** 输入目标岗位的 JD (Job Description) 或用户个人的修改诉求 (格式、内容、模板调整等)。
    *   **反馈功能:** AI 根据 JD 分析当前简历的不足，提供具有建设性、行业最优解的优化建议。
*   **中间 (代码/编辑区):** 展示简历的 HTML 源码。
*   **右侧 (实时预览区):** 实时渲染 HTML 代码，展示简历的最终效果。

### 3.2 自动化修改流程
1.  **AI 分析与建议:** 用户在左侧输入 JD 或诉求。
2.  **智能体反馈:** AI 分析后，向用户说明需要修改的地方及原因。
3.  **用户确认:** 用户确认修改建议。
4.  **自动执行:** AI 自动修改中间区域的 HTML 代码，右侧区域实时更新展示效果。
5.  **随时干预:** 用户可随时提出新的修改想法，进行格式、模板或内容的调整。

## 4. UI/UX 要求
- **风格:** 高级感、专业感，避免简陋。
- **语言:** 全局界面文字必须使用**中文** (遵循 2026-02-28 铁律)。
